# 🤖 AI Chat App

A modern AI-powered chat application built with Flask and vanilla JavaScript, featuring a refined UI, real-time interactions, and context-aware responses powered by Groq’s LLaMA models.


## ✨ Features

- ⚡ Fast AI responses powered by Groq’s LLaMA models  
- 💬 Real-time chat interface with smooth animations  
- 🧠 Persistent conversation memory (short-term + long-term)  
- 🎯 Clean, modern UI with glassmorphism design  
- 🌙 Adaptive dark/light mode  
- ⏳ Typing indicators for natural interaction  
- 📱 Fully responsive (mobile + desktop)  
- 🔐 Secure API key handling using `.env`  
- 🔄 Full frontend ↔ backend integration  
- 🧩 Scalable architecture for future features  


## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **AI:** Groq API (LLaMA models)  
- **Storage:** JSON-based memory system  


## 🚀 Overview

This project demonstrates the end-to-end implementation of an AI-powered chat system, combining:

- Clean UI/UX design  
- Real-time interaction handling  
- Context-aware AI responses  
- Persistent memory across conversations
